)
inlinestatic

Construct an SDIO CMD52 argument based on the given arguments.

Parameters
rwFlag indication direction (read or write).
fnThe applicable function.
addressThe address to read/write. Must be <= MMHAL_SDIO_ADDRESS_MAX.
write_dataThe data to write if this is a write operation. Should be set to zero for a read operation.
Returns
the SDIO CMD52 argument generated based on the given arguments.

enum mmhal_sdio_function  fn,

uint16_t  count 
)
inlinestatic

Construct an SDIO CMD53 argument based on the given arguments.

Parameters
rwFlag indication direction (read or write).
fnThe applicable function.
modeSelects between byte and block mode.
addressThe address to read/write. Must be <= MMHAL_SDIO_ADDRESS_MAX.
countThe count of bytes/blocks (depending on mode) to transfer. Must be <= MMHAL_SDIO_COUNT_MAX.
Note
OP Code 1 (incrementing address) is assumed. See also MMHAL_SDIO_OPCODE_INC_ADDR.
Returns
the SDIO CMD53 argument generated based on the given arguments.
