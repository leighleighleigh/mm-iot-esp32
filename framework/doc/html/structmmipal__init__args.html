mmipal_init_args Struct Reference
Morse Micro IP Stack Abstraction Layer (MMIPAL) API

Detailed Description

Initialize arguments structure.

This should be initialized using MMIPAL_INIT_ARGS_DEFAULT. For example:

struct mmipal_init_args args = MMIPAL_INIT_ARGS_DEFAULT;
MMIPAL_INIT_ARGS_DEFAULT
#define MMIPAL_INIT_ARGS_DEFAULT
