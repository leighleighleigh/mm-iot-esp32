Detailed Description

Iperf server arguments data structure.

For forward compatibility this structure should be initialized using MMIPERF_SERVER_ARGS_DEFAULT. For example:

struct mmiperf_server_args args = MMIPERF_SERVER_ARGS_DEFAULT;
